The Sabalcore terminal does not support Ctrl+V. Use Shift+Insert to paste in the terminal instead.
1
Pre-flight checklist
Mesh cleared in STAR-CCM+
File → Clear Generated Mesh before saving. Sabalcore handles meshing.
Max iterations set to 800
Stopping Criteria → Maximum Steps → 800
Convergence criterion enabled
Confirm drag criterion is active, not just defined
Force reports configured correctly
Drag, downforce, moments — correct reference area and direction
Sim saved as STAR-CCM+ 2506
Confirm version in title bar before transferring

Use this guide if you prefer submitting directly through the Sabalcore terminal.
1
Transfer files to Sabalcore

In ssClient go to Tools → SFTP File Transfers. Left pane is your PC, right pane is Sabalcore. Navigate to ~/bfr/ on the right. Transfer your .sim file and run.pbs.

2
Submit and monitor
cd ~/bfr
dos2unix run.pbs
qsub run.pbs
watch -n 5 qstat
3
Job states
Q Queued — waiting for nodes. No credits charged.
R Running — credits are being consumed.
C Complete — transfer results via SFTP from ~/bfr/.
4
Live output while running
tail -f ~/.pbs_spool/JOBID.jman.OU

Replace JOBID with your actual job number. Press Ctrl+C to stop.

5
Kill or clear
qdel JOBID.jman
rm -f ~/bfr/*
