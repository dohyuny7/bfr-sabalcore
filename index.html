Submission steps
1
Transfer files to Sabalcore
In ssClient go to Tools → SFTP File Transfers. Left pane is your PC, right pane is Sabalcore. Navigate to ~/bfr/ on the right. Transfer your .sim file and run.pbs.
2
Navigate to the job folder
cd ~/bfr
3
Fix Windows line endings
Always run this before submitting. The file was created on Windows and needs this fix.
dos2unix run.pbs
4
Submit the job
qsub run.pbs
You will get a job ID like 1477892.jman. Write it down.
5
Monitor the job
watch -n 5 qstat
Q queued, no charge   R running, credits burning   C complete

Press Ctrl+C to stop watching.
6
Watch live solver output while running
tail -f ~/.pbs_spool/JOBID.jman.OU
Replace JOBID with your actual job number. You will see residuals and force reports printing live. Press Ctrl+C to stop.
7
Check output after completion
Output files appear in ~/bfr/ after the job finishes.
cat ~/bfr/BFR_run.oJOBID
cat ~/bfr/BFR_run.eJOBID
The .e file should be empty if successful. The .o file has all residuals and force output.
8
Transfer results back to your PC
Use ssClient SFTP to pull _complete.sim and any results files back to your local machine for post-processing.
